# To Do Project

## Technologies Used

This project was built using the following technologies:

- Next.js
- Tailwind CSS

## Getting Started

First, install depedencies

```bash
yarn install
```

Then start developpement app:
```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
